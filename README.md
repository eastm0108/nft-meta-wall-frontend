# NFT META WALL

前端技術： `Vue 3`、`Imgur api`、`Vuex`、`vue-router`、`mockjs`

後端技術： `Node.js`、`Mongodb`、`JWT(未實作完)`

## 安裝套件

```bash=
npm install
```

## 新增 `.env.*` 檔案

範例格式：.env.development

```
NODE_ENV='development'
VITE_BASE_API='/mock/api/'
VITE_BASE_IMGUR_TOKEN='XXXXX'
VITE_BASE_IMGUR_ALBUM='XXXXX'
```
